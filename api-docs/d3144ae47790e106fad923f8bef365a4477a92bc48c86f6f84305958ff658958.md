* `path` {string|Buffer|URL}
* `flags` {string|number} 请参阅[文件系统标志的支持][support of file system `flags`]。
* `mode` {integer} 默认为 `0o666`。
* 返回: {number}

返回表示文件描述符的整数。

有关详细信息，请参阅此 API 的异步版本的文档：[`fs.open()`]。
